Documentation Fonctionnelle: Guide Utilisateur

Application d'Exploration DPE

1. Objectif de l'Application

En complément du rapport d'analyse statique, GreenTech Solutions met à votre disposition un tableau de bord interactif. Cet outil permet d'explorer les données de manière dynamique et de mener vos propres analyses exploratoires.

2. Accès et Connexion

L'application est protégée par un mot de passe pour garantir la confidentialité de l'analyse.

Utilisateur : greentech

Mot de passe : pas les meilleurs mais pas les pires non plus

3. Fonctionnalités Principales

L'application est structurée en plusieurs onglets, accessibles depuis le menu de gauche.

Onglet 1 : Comparaison Étiquettes

Cet onglet est l'écran principal. Il vous permet de :

Filtrer les données :

Type DPE : Isoler les logements "Neufs" ou "Existants", ou voir "Tous" les logements.

Département : Isoler le "59 - Nord" ou le "66 - Pyrénées-Orientales", ou voir les deux.

Année(s) : Sélectionner une plage d'années spécifique (par ex: 2022 à 2023).

Consulter les Indicateurs Clés (KPI) : Les trois boîtes (Logements, Passoires, Conso moyenne) se mettent à jour automatiquement en fonction de vos filtres.

Analyser la Distribution : Le graphique à barres principal montre la répartition des étiquettes DPE pour le périmètre que vous avez sélectionné.

Analyser l'Évolution : Le graphique en ligne du bas vous permet de choisir une étiquette DPE (ex: "F") et de voir son évolution (combien ont été enregistrés) mois par mois.

Onglet 2 : Variables Explicatives

Cet onglet vous aide à comprendre pourquoi un logement a une certaine note.

Top 3 Énergies de Chauffage : Montre les types de chauffage les plus courants pour chaque étiquette DPE.

Isolation des Murs : Un graphique 100% empilé qui montre la corrélation entre une mauvaise isolation et une mauvaise note DPE.

Corrélation Variables : Un outil avancé vous permettant de croiser deux variables numériques (par ex: "Surface habitable" vs. "Consommation") pour trouver des corrélations.

Onglet 3 : Carte DPE

Cet onglet affiche une carte interactive (via Leaflet) de tous les logements de votre sélection.

Les points sont colorés par étiquette DPE (du vert au rouge).

Vous pouvez zoomer et cliquer sur les "clusters" (groupes de points) pour voir les DPE dans une ville ou un quartier spécifique.

Onglet 4 : Données Brutes

Cet onglet affiche un tableau de données filtrées.

Vous pouvez trier les colonnes en cliquant sur les en-têtes.

Vous pouvez utiliser le bouton "Exporter CSV" pour télécharger une copie des données que vous avez filtrées.

Onglet 5 : Contexte

Rappelle les sources des données (ADEME).
